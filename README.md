## Api Integration

Random user api from [randomuser.me](https://randomuser.me/) 


## Tech Stack

**Frontend:** React.js,Tailwind CSS


## Run Locally

Clone the project

```bash
 https://github.com/Sravan-P-Thankappan/facePrep.git
```

Go to the project directory

```bash
  cd client 
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
 client : npm start
```
